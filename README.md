## 📖 About

EncryptX is a browser-based encryption project built using HTML, CSS, and JavaScript.

It provides a modern interface for experimenting with multiple encryption techniques, custom algorithm chains, AES-protected encrypted packages, and downloadable encryption keys.

---

## ✨ Features

- 🔐 Multiple encryption algorithms
- ⚙️ Custom algorithm builder
- 🛡️ AES-protected encrypted packages
- 🔑 Downloadable encryption keys
- 📦 Copy and download encrypted output
- 🌑 Full dark mode interface
- ✨ Interactive hover dropdown menus
- 🧠 Dynamic key generation
- 📝 Save and load custom presets
- 🎯 Tooltip-based guidance
- 📱 Responsive design

---

## 🧩 Supported Algorithms

| Algorithm | Description |
|---|---|
| Caesar Cipher | Shifts letters by a selected amount |
| XOR Cipher | Scrambles text using a key |
| Reverse Text | Reverses the entire message |
| Swap Case | Switches uppercase and lowercase |
| Base64 Layer | Encodes text into Base64 |
| ROT13 | Rotates letters by 13 |
| Atbash Cipher | Opposite alphabet mapping |
| Vigenere Cipher | Keyword-based shifting |

## ⚙️ Custom Builder

EncryptX includes a multi-step custom encryption builder where users can:

- ➕ Add encryption steps
- 🔄 Change algorithm order
- 🧠 Enter custom values
- 💾 Save presets
- 📂 Load presets

## 🛡️ Security Features

EncryptX uses:

- AES-GCM Encryption
- PBKDF2 Key Derivation
- SHA-256 Hashing
- Random Salt Generation
- Passcode Protected Packages
